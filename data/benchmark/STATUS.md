## SA sources (`sa_sources/`)

Source files used to build the `.osa` / `.oga` databases. Intermediate
extraction artefacts (per-chrom REVEL split, gnomAD tabix extracts,
SpliceAI/PhyloP intermediate VCFs) are regenerated by the build
scripts and were removed in v7 cleanup to free 12+ GB.

Downloaded by [`download_sa_sources.sh`](sa_sources/download_sa_sources.sh)
(see also [`benchmarks/README.md`](../../benchmarks/README.md) section 2).

| File | Status | Size | Used by | Source URL |
|------|--------|------|---------|------------|
| `clinvar.vcf.gz` | ✅ | 190 MB | `clinvar.osa` | [NCBI ClinVar GRCh38](https://ftp.ncbi.nlm.nih.gov/pub/clinvar/vcf_GRCh38/clinvar.vcf.gz) |
| `variant_summary.txt.gz` | ✅ | 419 MB | `clinvar_protein.oga` | [NCBI ClinVar variant_summary](https://ftp.ncbi.nlm.nih.gov/pub/clinvar/tab_delimited/variant_summary.txt.gz) |
| `gnomad.v4.1.constraint_metrics.tsv` | ✅ | 95 MB | `gnomad_genes.oga` | [gnomAD v4.1 constraint TSV](https://storage.googleapis.com/gcp-public-data--gnomad/release/4.1/constraint/gnomad.v4.1.constraint_metrics.tsv) |
| `revel-v1.3_all_chromosomes.zip` | ✅ | 637 MB | per-chrom `revel.osa` (regenerable) | [REVEL v1.3 (Sinai HPC mirror)](https://rothsj06.dmz.hpc.mssm.edu/revel-v1.3_all_chromosomes.zip) |
| `clinvar_2star_regions.bed` | ✅ | 800 KB | gnomAD tabix extraction (24,350 merged ranges) | derived from `clinvar_2star.vcf` via `bedtools merge -d 5000` |
| `clingen_gene_validity.csv` | ✅ | 1 MB | ClinGen GDV raw download | [ClinGen Gene-Disease Validity](https://search.clinicalgenome.org/kb/gene-validity/download) |
| `clingen_gdv.tsv` | ✅ | 1 MB | converted to genemap2 layout for `--source omim` | output of `clingen_gdv_to_oga.py` |

The per-chrom gnomAD exomes VCFs (used for `gnomad_chrN.osa`, plus the
distilled SpliceAI / PhyloP `.osa`) are tabix-extracted, not
downloaded in full — see [`build_gnomad_per_chrom.sh`](sa_sources/build_gnomad_per_chrom.sh).
The remote source is [`gs://gcp-public-data--gnomad/release/4.1/vcf/exomes/`](https://storage.googleapis.com/gcp-public-data--gnomad/release/4.1/vcf/exomes/).

Build scripts:

| Script | Purpose |
|--------|---------|
| [`download_sa_sources.sh`](sa_sources/download_sa_sources.sh) | one-shot download of all source files (ClinVar / gnomAD constraint / REVEL / ClinGen) |
| [`build_gnomad_per_chrom.sh`](sa_sources/build_gnomad_per_chrom.sh) | tabix-extract gnomAD v4 exomes per chrom + sa-build |
| [`build_spliceai_phylop.sh`](sa_sources/build_spliceai_phylop.sh) | distill SpliceAI + PhyloP from gnomAD INFO + sa-build |
| [`extract_spliceai_phylop_from_gnomad.py`](sa_sources/extract_spliceai_phylop_from_gnomad.py) | per-chrom extractor invoked by `build_spliceai_phylop.sh` |
| [`clingen_gdv_to_oga.py`](sa_sources/clingen_gdv_to_oga.py) | ClinGen GDV CSV → genemap2 TSV |

Run `bash benchmarks/check_urls.sh acmg` to re-verify all of the above
before a fresh build.
